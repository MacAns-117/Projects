1Q.) Which apps have the highest rating in the given available dataset?
sol) select App,Rating from playstore where Rating = (select max(Rating) from playstore);
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

2Q.) What are the number of installs and reviews for the above apps? Return the apps with the highest reviews to the top.
sol) select App,Installs,Reviews,Rating from playstore where Rating = ( select max(Rating) from playstore) order by Reviews desc;
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

3Q.) Which app has the highest number of reviews? Also, mention the number of reviews and category of the app.
sol) select App,Category,Reviews from playstore where Reviews = (select max(Reviews) from playstore) order by Reviews desc limit 1;
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

4Q.)What is the total amount of revenue generated by the google play store by hosting apps? (Whenever a user buys apps  from the google play store, the amount is considered in the revenue)
sol) select sum(Price*Installs) as Total_Revenue from playstore where Type = 'Paid';
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

5Q.) Which Category of google play store apps has the highest number of installs? also, find out the total number of installs for that particular category.
sol) select Category,sum(Installs) as Total_Installs from Playstore group by Category order by Total_Installs desc limit 1;
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

6Q.) Which Genre has the most number of published apps?
sol) select Genres,count(App) as Published_Apps from Playstore group by genres order by Published_Apps desc limit 1;

10Q.) Which is the best dating app? (Best dating app is the one having the highest number of Reviews)
sol) select App as Best_dating_App,Reviews from Playstore where Category = 'DATING' order by Reviews desc limit 1 ;

14Q.) Extract all negative sentiment reviews for Adobe Acrobat Reader with their sentiment polarity and sentiment subjectivity
